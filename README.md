    We provide a sample entropy similarity method based on nontargeted analysis results, which quantifies correlations between aquatic samples and helps identify pollution sources and contamination levels. The "similarity" calculation functions compute cosine similarity and tailored sample entropy to quantify the correlation between different water samples. The "process" module normalizes spectral intensity via log-transformation and generates a dictionary mapping SMILES strings to their normalized intensity values. The pipeline automatically exports quantitative results (based on SMILES intensity) to Excel files, supporting the identification of pollution sources and evaluation of pollution degrees.
